Laudo Fix

Projeto desenvolvido para uma empresa de tecnologia terceirizada ao DETRAN, onde foi identificada a necessidade de criar um programa que estabelecesse uma comunicação direta com o banco de dados. Essa aplicação possui uma série de validações essenciais antes de efetuar as atualizações diretamente no banco de dados. Essa abordagem permite que os atendentes façam as alterações necessárias com segurança e evita possíveis riscos.
